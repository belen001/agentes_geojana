# Agentes Geojana

Este es el repositorio principal para el sistema de agentes e integración de Geojana.

## Estructura del Proyecto

El proyecto está organizado en las siguientes carpetas:

*   **`fabrica_agentes_ia/`**: El núcleo de software de agentes de IA. Contiene el motor base reutilizable, la base de conocimiento portátil, las configuraciones por repositorio y las definiciones de los agentes.
    *   [Descripción Técnica de la Fábrica (10 Pilares)](file:///c:/Users/belen/Desktop/agentes_geojana-main/fabrica_agentes_ia/DESCRIPCION.md): Documento detallado sobre el funcionamiento del motor base en términos de *entrada clara*, *refinamiento*, *planificación*, *especialización por agentes*, *reglas y workflows*, *arnés*, *ejecución reproducible*, *validación*, *evidencia* y *cierre técnico*.
    *   [Guía de Uso Rápido (README)](file:///c:/Users/belen/Desktop/agentes_geojana-main/fabrica_agentes_ia/README.md): Instrucciones sobre cómo instalar, ejecutar tareas naturales en la CLI y configurar la base portable.
*   **`documentos/`**: Carpeta que almacena los requerimientos de diseño, ADRs (Architecture Decision Records) y pautas generales de diseño que alimentan el conocimiento portable de la fábrica de agentes.

## Cómo empezar

Para entender el diseño y arquitectura técnica de la fábrica de agentes de IA, lee el archivo [DESCRIPCION.md](file:///c:/Users/belen/Desktop/agentes_geojana-main/fabrica_agentes_ia/DESCRIPCION.md).

Para comenzar a interactuar con la línea de comandos de los agentes y ejecutar flujos de trabajo sobre Geojana u otros repositorios, consulta el archivo [README.md de la fábrica](file:///c:/Users/belen/Desktop/agentes_geojana-main/fabrica_agentes_ia/README.md).
