# Perfil backend

Enfoque:

- ADRs.
- Contrato de endpoints.
- Servicios.
- Modelo de base de datos.
- Autenticacion y autorizacion.

Politicas:

- Respetar ADRs antes de introducir tecnologia o patron nuevo.

- Citar decisiones tecnicas y de dominio con evidence_id.
- Ejecutar pruebas del servicio afectado cuando existan.
