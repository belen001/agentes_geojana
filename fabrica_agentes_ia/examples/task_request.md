# Solicitud

Generar un plan de implementacion para una funcionalidad nueva de Geojana usando la documentacion disponible.

## Alcance

- Respetar ADRs.
- Identificar servicios afectados.
- Marcar cualquier dato no encontrado como desconocido.

## Criterios de aceptacion

- El plan contiene pasos pequenos.
- Cada afirmacion de dominio usa citas de evidencia.
- La validacion de la corrida queda en estado `ok`.
